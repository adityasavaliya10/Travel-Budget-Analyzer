# Travel-Budget-Analyzer
Travel Budget Analyzer web app to estimate trip costs based on city, travel style, and duration. Built with focus on user flow, UI design, and logic, using online resources and AI assistance.

Note: This project does not use real-time APIs. The cost estimates are based on predefined values and are intended for general guidance only. Actual expenses may vary depending on individual preferences and travel conditions (approximate values for 2026).
The values shown are average estimates for each selected city.
